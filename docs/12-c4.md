# 12 — Diagramas C4

Modelo C4 (Contexto → Container → Componente). Diagramas em Mermaid.

## Nível 1 — Contexto
```mermaid
flowchart TB
    User([Usuário - iOS/Android])
    Admin([Suporte/Admin])
    subgraph Finlytics
      System[Finlytics Platform]
    end
    Quotes[(Provedor de Cotações)]
    Store[(App Store / Google Play - IAP)]
    Mail[(Provedor de E-mail)]
    Push[(FCM / APNs)]
    User -->|usa o app| System
    Admin -->|suporte/backoffice| System
    System -->|consulta preços| Quotes
    System -->|valida assinaturas| Store
    System -->|envia e-mails| Mail
    System -->|notificações| Push
```

## Nível 2 — Containers
```mermaid
flowchart TB
    App[Mobile App - Flutter]
    API[API - NestJS/Node]
    Worker[Workers - BullMQ]
    DB[(PostgreSQL)]
    Cache[(Redis)]
    Obj[(S3)]
    App -->|HTTPS/REST + JWT| API
    API --> DB
    API --> Cache
    API --> Obj
    API -->|enfileira jobs| Cache
    Worker -->|consome filas| Cache
    Worker --> DB
    Worker -->|cotações| Quotes[(Cotações)]
    Worker -->|e-mails| Mail[(E-mail)]
    API -->|push| Push[(FCM/APNs)]
```
